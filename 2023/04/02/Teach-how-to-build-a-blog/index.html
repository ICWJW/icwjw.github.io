利用 Obsidian 书写博客

  1. 在 Obsidian 里面选择 打开本地仓库 ，选择 Blog
  2. 设置 -> 文件和链接 -> 始终更新内部链接 -> 打开
  3. 设置 -> 文件和链接 -> 新建笔记的存放位置 -> 指定的附件文件夹
  4. 设置 -> 文件和链接 -> 内部链接类型 -> 基于当前笔记的相对路径
  5. 设置 -> 文件和链接 -> 忽略文件 -> node_modules 和 public 和 scaffolds 和 themes
  6. 在 source 新建文件夹 img ，设置 -> 文件和链接 -> 附件默认存放路径 -> 指定文件夹路径 source/img
  7. 设置 -> 模板 -> 模板文件夹位置 -> scaffolds
  8. 打开 scaffolds 下的 page 文件，设置自己的模板。
